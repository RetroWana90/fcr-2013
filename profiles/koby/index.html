My Video

From looking at all the different cell phone carriers in America, I noticed that they all are more or less very similar. From seeing all these similarities I wanted to put together a clip that proves how all of the major carriers advertise the same message over and over, whether it be in their: network speed, size, popularity, etc... This project is in NO way meant to glorify any of the companies shown. This project IS meant to show how similar the lies from these companies are.
